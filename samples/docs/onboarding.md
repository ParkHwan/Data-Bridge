## First week

New engineers set up workstation access on day one, pair with their onboarding buddy
on days two and three, and ship a starter task by the end of the week. The starter
task must touch the deploy pipeline so the release process is learned early.

## Required reading

Read the Deployment Runbook before your first on-call shadow. Read the Aurora Insights
product overview before joining sprint planning. Both live in the Handbook space.

## Access checklist

Request BigQuery read access to the analytics datasets, Cloud Storage access to the
dead-letter bucket, and dashboard editor rights in Pulse. Production write access is
granted only after the first month review.
