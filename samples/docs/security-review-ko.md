## 침투 테스트

외부 침투 테스트는 분기마다 1회, 외부 전문 업체를 통해 수행한다. 결과 보고서는
수령 후 5영업일 이내에 보안 채널에 공유한다.

## 취약점 대응 기한

심각(Critical) 등급 취약점은 발견 후 48시간 이내에 패치를 배포해야 한다.
높음(High) 등급은 7일, 중간(Medium) 등급은 30일 이내를 기한으로 한다.

## 접근 권한 검토

전체 시스템의 접근 권한을 매월 마지막 금요일에 검토한다. 90일 이상 미사용
계정은 자동으로 비활성화한다.

## 사고 보고

보안 사고를 인지한 담당자는 30분 이내에 보안 채널로 최초 보고를 올리고,
24시간 이내에 상세 타임라인을 작성한다.
